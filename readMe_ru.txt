Прототип проекта "Визуализатор состояния инфраструктуры"
Использовался для демонстрации отображения событий на инфраструктуре РЖД.

Основной функционал:

	Отображение различных типов событий на инфраструктуре.
	Задавать стили отображения событий.
	Задавать фильтры по событиям на инфраструктуре и сохранять их для последующего применения.
	Задавать группировки событий на инфраструктуре и сохранять их для последующего применения.
	Строить графики на таблицах событий.
	Строить отчеты на таблице событий (в т.ч. со сложными заголовками).
	Импортировать таблицы в exel файлы с сохранением заголовков,группировок и стилей.
	И т.д. )))

Демо-ролик (RU) доступен на https://cloud.mail.ru/public/DkQg/X4tPN2wxn

Для сборки проекта необходимо gwt-2.4.0 и smartgwt-5.0p
Для отображения графиков используется dojo 1.10.4
Пути библиотек должны быть отредактированы в build.properties build.xml.
Так же необходимо распаковать файл ./web/dojo_unpack_it_here.zip в папку ./web
